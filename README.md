# PDF面付け変換ツール (N-up Tool)

複数のPDFファイルを1枚のページに配置（面付け）するブラウザベースのツールです。
サーバーにアップロードすることなく、ローカルのブラウザ上で処理が完結するため、高速かつ安全に利用できます。

## 主な機能

- **面付け (N-up)**: 
  - 1x2 (2ページ/枚 - 縦)
  - 2x1 (2ページ/枚 - 横)
  - 2x2 (4ページ/枚)
  - 2x3 (6ページ/枚)
  - 3x3 (9ページ/枚)
  - 4x4 (16ページ/枚)
- **自動回転**: 横向きのページを自動的に回転させて向きを統一します。
- **個別回転**: ファイルリストから個別にファイルの向きを90度ずつ回転できます。
- **個別削除**: ファイルリストから個別にファイルを削除できます。
- **プレビュー機能**: 変換前にレイアウトを確認できます。
- **ダークモード対応**: システム設定に合わせて、または手動でダークモードに切り替えられます。
- **モバイル対応**: iPhone/iPadでもファイルを1つずつ追加して処理できます。

## 使い方

1. **ファイルを追加**: PDFファイルをドラッグ＆ドロップするか、「ファイルを追加」ボタンから選択します。
   - デスクトップ（Chrome/Edge）ではフォルダごとの読み込みも可能です。
   - モバイルでは1ファイルずつ追加できます。
2. **設定**: 
   - レイアウト（面付け数）を選択します。
   - 必要に応じて余白やセル間隔を調整します。
   - 「ページの向きを自動で揃える」などのオプションを設定します。
   - 必要であれば、ファイルリストの回転ボタンで個別のファイルの向きを調整します。
3. **実行**: 「PDF変換を実行」ボタンを押すと、処理が開始され、完了すると自動的にダウンロードされます。

## 対応ブラウザ

- Safari（macOS / iOS）
- Chrome / Edge（デスクトップではフォルダ選択も可能）
- Firefox

## トラブルシューティング

### 「対応していない圧縮形式」エラーが出る場合
macOSの「プレビュー」でPDFを開き、「ファイル」→「書き出す」で再保存してから再度お試しください。

### 「パスワードで保護されています」エラーが出る場合
パスワードが必要なPDFは処理できません。パスワードを解除してから再度お試しください。
