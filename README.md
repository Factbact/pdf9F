# PDF ツール集

複数のPDFファイルを操作するブラウザベースのツール集です。
サーバーにアップロードすることなく、ローカルのブラウザ上で処理が完結するため、高速かつ安全に利用できます。

## ツール一覧

### 1. PDF面付けツール (index.html)
複数のPDFページを1枚に配置（面付け）します。

- **レイアウト**: 1x2, 2x1, 2x2, 2x3, 3x3, 4x4
- **自動回転**: 横向きページを自動で揃える
- **個別回転**: ファイルごとに90度ずつ回転
- **プレビュー**: 変換前にレイアウトを確認

### 2. PDF結合ツール (merge.html)
複数のPDFファイルを1つに結合します。

- **ドラッグ並び替え**: ファイルの順番を自由に変更
- **個別回転**: ファイルごとに90度ずつ回転
- **アウトライン自動追加**: 各ファイルにブックマーク（しおり）を作成
- **ファイル名自動生成**: 最初と最後のファイル名から自動生成

## 使い方

1. ブラウザで `index.html`（面付け）または `merge.html`（結合）を開く
2. PDFファイルをドラッグ＆ドロップまたはボタンから追加
3. 必要に応じて設定を調整
4. 実行ボタンをクリック

## 対応ブラウザ

- Safari（macOS / iOS）
- Chrome / Edge（デスクトップではフォルダ選択も可能）
- Firefox

## トラブルシューティング

### 「対応していない圧縮形式」エラー
macOSの「プレビュー」でPDFを開き、「ファイル」→「書き出す」で再保存してください。

### 「パスワードで保護されています」エラー
パスワードが必要なPDFは処理できません。

## 技術スタック

- JavaScript (Vanilla JS)

- [PDF.js](https://mozilla.github.io/pdf.js/): プレビュー・サムネイル
