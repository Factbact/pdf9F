# PDF Web Tools

ブラウザ上で動作する、安全で便利なPDF・画像処理ツール群です。
すべての処理はクライアントサイド（ブラウザ内）で行われるため、ファイルがサーバーにアップロードされることはなく、プライバシーが守られます。

---

## 搭載ツール（全7種）

### 1. PDF面付け — `index.html`
複数のPDFページを1枚の用紙に割り付ける（N-up印刷）。
- 1x2 / 2x1 / 2x2 / 3x3 / 4x4 レイアウト対応
- 自動回転・手動回転、出力用紙サイズ指定（A3 / A4 / B5 / Auto）
- アウトライン（しおり）の自動付与オプション

### 2. PDF結合 — `merge.html`
複数のPDFファイルをまとめて1つのPDFに結合する。
- ドラッグ&ドロップで順序の並び替え
- アウトライン（しおり）の自動付与オプション
- 出力ファイル名のカスタマイズ

### 3. PDF分割 — `split.html`
1つのPDFを複数のファイルに分割する。
- ページ範囲指定による分割
- 1ページずつ個別ファイルへの分割

### 4. PDF圧縮 — `compress.html`
PDFファイルのサイズを縮小する。
- 圧縮品質の選択
- 圧縮前後のサイズ比較表示

### 5. HEIC変換 — `heic.html`
iPhoneなどで撮影されたHEIC / HEIF画像を他の形式に変換する。
- 出力フォーマット選択: **PNG / JPEG / WebP**
- JPEG選択時は品質スライダー（1〜100）で画質調整
- 複数ファイル一括変換対応
- ライブラリ: `libheif-js`（WebAssembly版）を使用し、iPhone 12以降の10bit HEICにも対応

### 6. 画像→PDF変換 — `img2pdf.html`
JPG / PNG / GIF / WebP / HEICの画像をまとめてPDFに変換する。
- ページサイズ設定（A4 / A3 / 画像に合わせる）
- 余白・配置のカスタマイズ

### 7. アウトライン並び替え — `outline.html`
PDFのアウトライン（しおり）を自然順（数値順）に並び替えてダウンロードする。
- `localeCompare({ numeric: true })` による自然順ソート（1, 2, 3 ... 10, 11）
- ネストした階層も再帰的にすべてソート
- ソート前・ソート後のプレビューを並列表示してから確認してダウンロード
- 宛先（Dest / Action）・装飾（色・スタイル）・展開/折りたたみ状態を保持

## 主な特徴

- **完全クライアントサイド**: ファイルはブラウザ内のみで処理。サーバー送信なし
- **ダークモード対応**: OSの設定に連動、手動切り替えも可
- **日本語 / 英語** の多言語対応（切り替えボタン付き）
- **レスポンシブデザイン**: PC / タブレット / スマホ対応

## 既知の問題

- iOS / iPadOSの一部ブラウザで、PDF回転処理が正しく反映されない場合がある（PCでの利用を推奨）
- 非常に大きなPDF（100MB超）はメモリ不足でブラウザがクラッシュすることがある
