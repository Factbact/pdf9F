# PDF面付け変換ツール (N-up Tool)

複数のPDFファイルを1枚のページに配置（面付け）するブラウザベースのツールです。
サーバーにアップロードすることなく、ローカルのブラウザ上で処理が完結するため、高速かつ安全に利用できます。

## 主な機能

- **面付け (N-up)**: 
  - 1x2 (2ページ/枚 - 縦)
  - 2x1 (2ページ/枚 - 横)
  - 2x2 (4ページ/枚)
  - 2x3 (6ページ/枚)
  - 3x3 (9ページ/枚)
  - 4x4 (16ページ/枚)
- **自動回転**: 横向きのページを自動的に回転させて向きを統一します。
- **個別回転**: ファイルリストから個別にファイルの向きを90度ずつ回転できます。
- **プレビュー機能**: 変換前にレイアウトを確認できます。
- **ダークモード対応**: システム設定に合わせて、または手動でダークモードに切り替えられます。

## 使い方

1. **ファイルを選択**: PDFファイルをドラッグ＆ドロップするか、「ファイルを選択」ボタンから選択します。フォルダごとの読み込みも可能です（Chrome/Edge等）。
2. **設定**: 
   - レイアウト（面付け数）を選択します。
   - 必要に応じて余白やセル間隔を調整します。
   - 「ページの向きを自動で揃える」などのオプションを設定します。
   - 必要であれば、ファイルリストの回転ボタンで個別のファイルの向きを調整します。
3. **実行**: 「PDF変換を実行」ボタンを押すと、処理が開始され、完了すると自動的にダウンロードされます。

## 技術スタック

- HTML5 / CSS3 (Tailwind CSS)
- JavaScript (Vanilla JS)
- [pdf-lib](https://pdf-lib.js.org/): PDF操作ライブラリ
