# PDF N-up Tool (PDF面付け変換ツール)

PDFファイルを面付け（N-up）形式に変換するWebツールです。

## 機能

- **レイアウト選択**: 1x2, 2x1, 2x2, 2x3, 3x3, 4x4 などのレイアウトに対応
- **自動回転**: ページの向きを自動で揃える
- **手動回転**: 全ページを一括で回転
- **個別回転**: ファイルごとに回転を設定
- **プレビュー機能**: 変換前にレイアウトを確認
- **バッチ処理**: 複数ファイルの一括変換
- **多言語対応**: 日本語・英語

## 使用方法

1. PDFファイルをドラッグ＆ドロップまたは選択
2. レイアウトや余白などの設定を調整
3. 「PDF変換を実行」ボタンをクリック

## 既知の問題

### iPad/iOSデバイスでの動作について

- **問題**: iPadなどのiOSデバイスで実行した場合、ページの回転（手動回転・個別ファイル回転）が正しく適用されない場合があります。
- **回避策**: PCのブラウザ（Chrome, Firefox, Edge など）での使用を推奨します。
- **原因**: iOSのSafariやWebKitベースのブラウザでは、PDFの回転処理に制限がある可能性があります。
