# PDF Web Tools

ブラウザ上で動作する、安全で便利なPDF・画像処理ツール群です。
すべての処理はクライアントサイド（ブラウザ内）で行われるため、ファイルがサーバーにアップロードされることはなく、プライバシーが守られます。

## 🛠 搭載ツール

### 1. PDF面付け (N-up)
- 複数のPDFページを1枚の用紙に割り付け（2in1, 4in1など）
- **特徴**: 自動回転、手動回転、レイアウト選択、出力用紙サイズ指定（A3/A4/B5/Auto）

### 2. PDF結合 (Merge)
- 複数のPDFファイルを1つのファイルに結合
- **特徴**: ドラッグ&ドロップで順序並べ替え、簡単操作

### 3. HEIC to PNG 変換
- iPhoneなどで撮影されたHEIC画像を、汎用的なPNG形式に変換
- **特徴**: 複数ファイル一括変換、ドラッグ&ドロップ対応

### 4. 画像 → PDF 変換 (Image to PDF)
- JPG, PNG, GIF, WebP, HEIC画像をまとめて1つのPDFに変換
- **特徴**: ページサイズ設定（A4/A3/Fit）、余白設定、配置（中央/Fit）

## ✨ 主な機能・特徴

- **完全オフライン動作**: サーバーへのアップロードなし（セキュリティ安心）
- **モダンなUI**: Tailwind CSS による使いやすく美しいデザイン（ダークモード対応）
- **多言語対応**: 日本語 / 英語
- **レスポンシブ**: PC、タブレット、スマホに対応

## 💻 技術情報

- PDF操作: `pdf-lib`, `pdf.js`
- 画像変換: `heic2any`
- スタイリング: `Tailwind CSS`
- フォント: `Plus Jakarta Sans`, `Outfit`

## ⚠️ 既知の問題
- iOS/iPadOSの一部のブラウザで、PDFの回転処理が正しく反映されない場合があります。PCでの利用を推奨します。
